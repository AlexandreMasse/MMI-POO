jQuery : Afficher/Cacher le mot de passe

L'évènement permettant d'écouter un changement sur un élément de formulaire s'appelle change. Il se déclanche lorsque l'utilisateur modifie directement la valeur d'un champs.

Afin de vérifier si une "checkbox" est cochée ou non, vous pouvez utiliser la fonction jQuery .is()
Par exemple, pour vérifier si un élément est un <li>, on peut écrire : if ($('.element').is('li'))
Pour vérifier si un élément est le dernier enfant de son parent, on peut aussi utiliser la pseudo-classe :last-child comme en CSS : if ($('.element').is(':last-child'))
Pour info, il existe une pseudo-classe nommée :checked
