jQuery : Menu déroulant

Le $(this) en jQuery s'utilise à l'intérieur d'une fonction évènementielle. Il représente l'objet jQuery qui a lancé l'évènement.

Ainsi, dans le code suivant :


  $('a').on('click', function () {
    alert( $(this).attr('href') );
  });

... le $(this) représente uniquement le lien <a> qui a été cliqué (et non tous les <a>)
